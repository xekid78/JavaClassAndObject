# JavaClassAndObject
クラスとオブジェクト

## 処理
`Massage()`クラスの`HelloWorld()`メソッドを使う。

## コード
```
public class Clsandobj {

	public static void main(String[] args) {
		Message msg = new Message();
		msg.HelloWorld();

	}

}

class Message {

	public void HelloWorld() {
		System.out.println("Hello World");

	}

}
```

## 出力結果
```
Hello World
```
